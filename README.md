# 京东晒单及评价查看大图

## 晒单查看大图

进入网页版**商品评价**，进入**晒单**选项，当鼠标移入下方预览图时，会在右上角出现一个**查看大图**按钮。点击后会在新页面打开当前图片的大图。

![晒单查看大图](https://raw.githubusercontent.com/ssyatelandisi/JDShaidanBigImage/master/01.jpg)

------

## 评价查看大图

进入网页版**商品评价**，进入**全部评价**选项，当鼠标移入下方预览图时，会在右上角出现一个**查看大图**按钮。点击后会在新页面打开当前图片的大图。

![评价查看大图](https://raw.githubusercontent.com/ssyatelandisi/JDShaidanBigImage/master/02.jpg)
